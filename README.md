# 🚀 English Adventures

**تطبيق تعلم الإنجليزية بالعيش فيها**

## 🎯 المميزات

- 📖 **قصص تفاعلية** يومية بأسلوب أنمي
- 🧠 **AI Memory** — ذاكرة شخصية تتعلم من أخطائك
- 🌍 **World Mode** — مدينة افتراضية حية
- 🎙️ **نطق ذكي** مع تقييم AI
- 🏆 **Gamification** — نقاط، شارات، لوحة متصدرين
- 📸 **Life Scanner** — صور أي شيء، تعلم كلماته

## 🛠️ التقنية

- React 18
- Tailwind CSS
- Framer Motion
- Web Speech API
- PWA (Progressive Web App)

## 🚀 التشغيل

```bash
npm install
npm start
```

## 📱 الرفع على Vercel

```bash
npm install -g vercel
vercel
```
